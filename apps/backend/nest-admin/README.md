# Project setup

```bash
pnpm install
```

## Compile and run the project

```bash
# development
pnpm run dev

# watch mode
pnpm run dev:watch

# production mode
pnpm run dev:prod
```

## Run tests

```bash
# unit tests
pnpm run test

# e2e tests
pnpm run test:e2e

# test coverage
pnpm run test:cov
```
